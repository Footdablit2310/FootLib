# 📄 Create Train Fares — Project Status
Create Train Fares is currently on indefinite delay.
This mod depends heavily on the CRN (Create Railways Navigator) API for accessing train data, stations, schedules, and routing. At this time, CRN does not provide a stable or documented API for external mods to integrate with. Because of this, any implementation would be fragile, break every update, and require constant rewrites.
Rather than ship an unstable or incomplete system, development is paused until CRN releases an official, stable API that supports:
- station and line metadata
- train and schedule access
- routing and fare‑calculation hooks
- safe external integration
Once CRN exposes these features, Create Train Fares will resume development with a clean, maintainable architecture.
For now, this repository remains intentionally empty.
